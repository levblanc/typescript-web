# typescript-web

Implemented a simple web framework with Model & View.

## Quick Start

```zsh
# Install dependencies
npm i

# Run web view
npm run start:parcel
# Web view is up at: http://localhost:1234

# Install server
npm i -g json-server

# Run server
npm run start:db
# Server is up at: http://localhost:3000
```
